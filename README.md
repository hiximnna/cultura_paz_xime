# Cultura de Paz - Prevención de la violencia escolar

Tema: Violencia escolar  
Este proyecto tiene como objetivo fomentar una **cultura de paz dentro del entorno escolar**, promoviendo el respeto, la empatía y la comunicación positiva entre estudiantes, docentes y comunidad educativa.  
A través de una página web informativa y un cartel digital con el mensaje *"El respeto construye la paz"*, se busca crear conciencia sobre las consecuencias del acoso, la exclusión y otras formas de violencia.  
La página incluye **señales de alerta**, **acciones de autocuidado** y **rutas de apoyo** confiables para estudiantes que enfrentan situaciones de riesgo. Además, proporciona enlaces a instituciones oficiales que ofrecen orientación y acompañamiento en casos de violencia.  
El propósito principal es fortalecer los valores de convivencia pacífica, respeto y solidaridad, impulsando la creación de entornos educativos seguros e inclusivos.
